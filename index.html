Apa itu Web Development

Web development adalah proses pembuatan dan pemeliharaan situs web. Ini melibatkan berbagai tahap, termasuk perancangan tampilan (Front End) dan pengembangan fungsionalitas (Back End), serta integrasi berbagai teknologi seperti HTML, CSS, Javascript, dan server side scripting. Tujuannya adalah menciptakan situs web yang fungsional, interaktif, dan sesuai dengan kebutuhan pengguna.

Web Development

Web development dibagi menjadi tiga bagian utama : Front End development, Back End development, dan Fullstack development.

Front End

Ini melibatkan pembuatan tampilan visual situs web yang dilihat oleh pengguna. Front End developers bekerja dengan HTML, CSS, dan JavaScript untuk merancang halaman web yang menarik dan responsif.

Back End

Fokusnya adalah pada pengembangan komponen yang tidak terlihat oleh pengguna, seperti database, server, dan logika bisnis. Back End developers menggunakan bahasa pemrograman seperti Python, PHP, Ruby, atau Node.js untuk mengelola data dan menjalankan fungsionalitas situs.

Fullstack

Ini menggabungkan keterampilan Front End dan Back End development. Seorang Fullstack developer mampu mengatasi seluruh proses pembuatan situs web dari awal hingga akhir. Ketiga komponen ini bekerja bersama untuk menciptakan situs web yang lengkap, fungsional, dan menarik bagi pengguna.

Front End

  1. HTML (HyperText Markup Language)
    Digunakan untuk struktur dasar halaman web, seperti teks, gambar, tautan, dan elemen lainnya.
  2. CSS (Cascading Style Sheets)
    Digunakan untuk mengatur tampilan visual halaman web, termasuk warna, tata letak, jenis huruf, dan animasi.
  3. JavaScript
    Bahasa pemrograman yang memungkinkan interaksi dinamis dengan pengguna. Dengan JavaScript, Anda bisa membuat elemen interaktif seperti tombol klik, tampilan gambar, atau validasi formulir.
  4. Framework
    Contohnya adalah React, Angular, Vue.js, Laravel, dan CodeIgniter. Framework ini membantu dalam mengelola struktur kompleks dan pengembangan antarmuka pengguna yang lebih interaktif.
  5. Libraries
    Seperti jQuery, yang mempermudah tugas-tugas umum dalam pengembangan Front End, seperti manipulasi DOM dan pengiriman permintaan HTTP.

Back End

  1. Server
    Komputer atau sistem yang menjalankan aplikasi dan menyediakan data kepada pengguna. Contohnya adalah Apache, Nginx, atau Microsoft IIS.
  2. Database
    Tempat penyimpanan data, seperti informasi pengguna, produk, atau konten. Contoh database termasuk MySQL, PostgreSQL, MongoDB, dan SQLite.
  3. API (Application Programming Interface)
    Antarmuka yang memungkinkan berbagai aplikasi berkomunikasi dan bertukar data. Back End developers sering membuat API untuk menghubungkan Front End dengan database atau layanan lainnya.
  4. Authentication and Authorization
    Mekanisme untuk mengamankan akses ke data dan fitur tertentu. Contohnya adalah Auth, JWT (JSON Web Tokens), dan mekanisme autentikasi kustom.
  5. Security
    Mengamankan data dan melindungi aplikasi dari serangan, seperti serangan SQL injection atau cross-site scripting (XSS).
  6. Server Side-Scripting
    Menghasilkan konten dinamis yang akan ditampilkan di Front End, seperti mengambil data dari database dan menyajikannya dalam bentuk halaman web.

Fullstack

  1. Front End

    - Menggunakan HTML, CSS, dan JavaScript untuk merancang tampilan situs.

    - Menggunakan React, Angular, atau Vue.js untuk membangun antarmuka pengguna yang dinamis dan interaktif.

  2. Back End

    - Menggunakan Node.js dengan framework Express untuk mengembangkan server.

    - Menggunakan MongoDB sebagai database untuk menyimpan data.

  3. API
    Membuat API dengan Express untuk menghubungkan Front End dengan Back End.
  4. Deployment
    Deploy situs web menggunakan layanan seperti Heroku atau Netlify.
  5. Testing
    Melakukan pengujian unit dan integrasi pada baik Front End maupun Back End.
